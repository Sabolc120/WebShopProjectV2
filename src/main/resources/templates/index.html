Üdvözöllek a főoldalon, ez itt egy hobbi projekt

A projekt a következő technológiákat használja:

  • Spring, Java
  • Jpa alapismeretek, Hibernate
  • Lokális adatbázis MySql
  • Frontendként Bootstrap, html, css, Thymleaf

A weboldalon használt képek nincsenek a tulajdonomban, és a termék adatok kizárólag kitaláció.
